This is some tools you need to create good Micro Lua games.

- GBA Graphics to Micro Lua Map Converter
This tool convert files.map.c generated by GBA Graphics into .map used by Micro Lua.

- Lua compiler
This tool compile your .lua files into compiled lua files.

- no$gba Tester
This tool is very useful to test your scripts on the no$gba emulator

- OSLfonts
This tool allows you to convert your system (Windows) fonts into files used by Micro Lua.

- Soundbank Maker
This tool allows you to create sound banks to use in Micro Lua scripts
